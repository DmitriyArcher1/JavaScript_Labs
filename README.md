## 🚀 Лабораторная работа №1

### Пенсионный калькулятор

Калькулятор для определения возраста выхода на пенсию с учётом пола пользователя.

**Функциональность:**
- Ввод возраста
- Выбор пола (мужчина/женщина)
- Вывод результата с рекомендацией

**Результаты:**
- 0–17 лет → "Вам работать ещё рано — учитесь"
- 18–59 (муж) / 18–54 (жен) → "Вам ещё работать и работать"
- 60–64 (муж) / 55–59 (жен) → "Скоро пенсия!"
- 65+ (муж) / 60+ (жен) → "Вам пора на пенсию"

**Стек:** HTML, CSS, JavaScript

## 🚀 Лабораторная работа №2

### Базовые функции JavaScript

Реализация 8 базовых функций с документацией JSDoc и тестированием.

| Функция | Описание | Пример |
|---------|----------|--------|
| `pow(x, n)` | Возведение в степень | `pow(2, 3) → 8` |
| `sumTo(n)` | Сумма чисел от 1 до n | `sumTo(100) → 5050` |
| `isLeapYear(year)` | Проверка високосного года | `isLeapYear(2024) → true` |
| `factorial(n)` | Факториал числа (BigInt) | `factorial(5) → 120n` |
| `fib(n)` | Числа Фибоначчи (BigInt) | `fib(100) → 354224848179261915075n` |
| `compare(x)` | Функция сравнения | `compare(5)(6) → true` |
| `sum(...args)` | Сумма аргументов | `sum(1, 2, 3) → 6` |
| `addBlackSpot(obj)` | Добавление символа | `addBlackSpot({}) → {[Symbol.for("blackSpot")]: true}` |

**Стек:** JavaScript, Mocha, Chai, JSDoc

## 🚀 Лабораторная работа №3

### Работа со строками и числами

Набор функций для обработки строк, чисел и массивов.

| Функция | Описание | Пример |
|---------|----------|--------|
| `getDecimal(num)` | Дробная часть числа | `getDecimal(1.23) → 0.23` |
| `divmod(a, b)` | Деление с остатком | `divmod(7, 3) → [2, 1]` |
| `ucFirst(str)` | Заглавная буква | `ucFirst('hello') → 'Hello'` |
| `normalizeUrl(url)` | Нормализация URL | `normalizeUrl('yandex.ru') → 'https://yandex.ru'` |
| `checkSpam(str)` | Проверка спама | `checkSpam('_XxX_') → true` |
| `truncate(str, n)` | Обрезка строки | `truncate('Мама мыла раму.', 10) → 'Мама мыла…'` |
| `camelize(str)` | Преобразование в camelCase | `camelize('var-test-text') → 'varTestText'` |
| `fibs(n)` | Массив чисел Фибоначчи | `fibs(5) → [0n, 1n, 1n, 2n, 3n]` |
| `arrReverseSorted(arr)` | Сортировка по убыванию | `arrReverseSorted([1, 3, 22]) → [22, 3, 1]` |
| `unique(arr)` | Уникальные значения | `unique([0, 1, 1, 2]) → [0, 1, 2]` |

**Стек:** JavaScript (ES Modules), JSDoc, Mocha, Chai
